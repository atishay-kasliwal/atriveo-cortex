# Reports & Audits

Historical build reports, audits, and architecture notes generated during Cortex
development. Reference material — not living docs. Current design docs live one level
up in [`docs/`](../).

Files are named by area: `*_REPORT.md`, `*_AUDIT.md`, `*_ARCHITECTURE.md`,
`*_ROOT_CAUSE.md`. Two are regenerated by audit scripts:
`CONVERSATIONAL_CORTEX_AUDIT_V2.{md,json}` and `PROJECT_ATTRIBUTION_BASELINE.md`.
